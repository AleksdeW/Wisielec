# Wisielec
Projekt zespołowy – gra Wisielec realizowana w ramach zajęć
1. **Aleksander de Wolf** - Zaprojektowanie mechaniki trybu wieloosobowego, obsługa przejść w menu głównym oraz sterowanie ogólnym przepływem gry (np. wywoływanie pytania o zapis po zakończonej partii).
2. **Jan Fuksiewicz** - Opracowanie "silnika" wisielca – weryfikacja wpisywanych liter, aktualizacja liczby błędów (żyć) oraz sprawdzanie warunków wygranej lub przegranej.
3. **Magda Dziużanska** - Wyświetlanie szubienicy (ASCII art) w zależności od liczby błędów, estetyczne rysowanie planszy i maskowanie hasła, oraz bezpieczne pobieranie znaków od użytkownika.
4. **Mateusz Dalida** - Stworzenie bazy haseł i funkcji losującej, a także obsługa zapisu i odczytu stanu gry oraz statystyk z plików zewnętrznych.
